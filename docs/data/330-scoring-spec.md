# Scoring Spec v1

## Universal Weights (v1.0)
- Repairability: 25
- Company Ethics: 25
- Environment: 25
- Circularity: 25

## Modifiers
- Category overlays may adjust weights slightly (documented per overlay)

## Output
- `score_total` [0..100]
- `breakdown[]` with per-dimension contributions
- `provenance_coverage` and `confidence`
